Quick Start

推荐操作顺序

1. 先注册用户账号

每个人先创建自己的独立用户 ID，例如 `father_001`、`daughter_001`。

2. 给对象指定所属人

被监护对象可以归属给某个用户，系统会按这个归属关系去找紧急联系人。

3. 绑定联系人 ID

例如父亲账号下面绑定女儿账号。之后只要父亲对应对象告警，就会自动分发给女儿。
