Tips

房间内判断标准

1. 看共享后端

主机页通常显示本机地址，第二台电脑应显示主机的局域网地址。两边必须指向同一个后端。

2. 看会话 ID

两边房间页右上角的会话 ID 必须完全一样，否则一定不是同一场通话。

3. 看房间参与者

当参与者列表里同时出现家属端和设备端，远端数量变成 1 时，就说明两边进到了同一场通话。
