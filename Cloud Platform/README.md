软件运行代码汇总说明

本汇总包按运行角色分为 3 个顶层目录：

1. cloud-integration
   华为云接入相关代码与配置模板。这里不包含独立运行在华为云上的应用工程，主要是本项目中与 Huawei IoTDA webhook、配置模板和接入测试有关的代码。

2. mini-program
   微信小程序前端代码，包含页面、全局逻辑、样式和请求工具。

3. backend-service
   FastAPI 后端服务主代码，包含事件解析、数据库、微信订阅消息、静态 Web 演示页、测试以及附加客户端代码。

说明：
- 出于安全考虑，未包含 .env、数据库文件、日志、虚拟环境和构建产物。
- 如需部署，请结合 .env.example 自行填写环境变量。
